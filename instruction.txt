================================================================================
0. OVERALL GOAL
================================================================================

This experimental protocol is designed to test two central hypotheses in SiT:

H1. There exists a common representation shared across layers and timesteps.
H2. That shared/common representation mainly encodes spatial structure, while the residual after subtracting the common part retains information specific to layer identity and/or timestep identity.

To test these hypotheses, the protocol must:
- extract activations A_{i,t}(x) at each layer i and timestep t;
- also include A_patch(x), the feature after patchify / patch embedding before entering the model;
- define A_common in multiple ways:
  (a) mean-common
  (b) subspace-common, in practice fixed-rank truncated-SVD common for the main protocol
- define the residual as:
    B_{i,t}(x) = A_{i,t}(x) - A_common(x)
  or, for the subspace case:
    B_{i,t}(x) = A_{i,t}(x) - Proj_{U_common}(A_{i,t}(x))

Then evaluate the representations using:
- basic statistics: min, max, mean, variance
- visualization: PCA-RGB, t-SNE, UMAP
- linear probes: semantic probe (if labels exist), layer-id probe, timestep-id probe
- similarity metrics: token-mean cosine, flattened cosine, CKA
- patch-wise cosine maps
- spatial metrics: LDS, CDS, SRSS, RMSC
- spatial controls / stress tests
- wavelet decomposition

The main experiments kept in the final protocol are:
- Exp. 1: causal ablation
- Exp. 2: multiple common definitions
- Exp. 3: spatial stress test
- Exp. 6: spatial metrics / spatial probe family
- Exp. 7: layer-timestep map
- Exp. 10: wavelet decomposition

1. GLOBAL DEFAULT SETTING
================================================================================

1.1 Backbone / model setting

Use SiT-XL/2 as the default backbone for the main experiment.

Fixed model setting:
- Model size: SiT-XL/2
- Number of transformer blocks: L = 28
- Hidden dimension: D = 1152
- Number of attention heads: 16
- Patch size: 2
- Main image resolution: 256 x 256

Compact notation:
SiT-XL/2 : (L, D, H_head) = (28, 1152, 16)

All activation tensors should follow:
A_{i,t}(x) in R^{P x 1152}, for i in {1,...,28}, t in T
where P is the number of patch tokens in the latent grid.

1.2 Timestep setting

Use 10 representative timesteps.

Let the full scheduler / trajectory length be T_total.
Define:
T = {t_1, ..., t_10}

Uniform selection:
t_k = floor( ((k-1)/9) * (T_total - 1) ),  for k = 1,...,10

Rationale:
- sparse enough to capture early / middle / late behavior
- small enough to control storage/computation cost
- large enough so timestep heatmaps remain meaningful

For the first official run, use exactly the evenly spaced 10 timesteps above.

1.3 Layer setting

Preferred setting:
- use all 28 layers

If storage/computation becomes too heavy, a reduced exploratory setting may use 8-12 representative layers:
- 2-3 early layers
- 3-4 middle layers
- 2-3 late layers

However, because the central hypothesis explicitly concerns common structure across layers, the main protocol should use all 28 layers.

1.4 Dataset size

Pilot run:
- 100 images
- 10 timesteps
- all layers
- 1 checkpoint
- 1 seed

Main run:
- 500 to 1000 images
- 10 timesteps
- all layers
- 1 main checkpoint
- 1 seed

Recommended for the official main result:
- main dataset: 500 images
- control dataset: 100 images

1.5 Seed

Use exactly 1 main seed for the official protocol.
Do not mix multiple seeds inside the core protocol.

1.6 Storage / dtype

Do not keep the entire activation bank on GPU.
Write tensors to disk batch-by-batch or shard-by-shard.

Recommended storage style:
- activations/img_{id}/t_{t}/layer_{i}.pt
or
- shard_000.pt, shard_001.pt, ...

Recommended dtypes:
- bfloat16 or float16 for general tensor storage
- float32 for PCA / SVD / CKA / wavelet computations

1.7 Metadata

Every saved tensor bank must include metadata with:
- image id
- timestep
- layer
- seed
- checkpoint
- dtype
- common variant name

================================================================================
2. CORE TENSORS AND DEFINITIONS
================================================================================

2.1 Raw activation tensor

For each image x, timestep t, and layer i:
A_{i,t}(x) in R^{P x D}

Also extract:
A_patch(x)

2.2 Per-token L2 normalization

Before constructing common representations, normalize each patch token:

A_tilde_{i,t}(x)_p = A_{i,t}(x)_p / ( ||A_{i,t}(x)_p||_2 + eps )

Use:
eps = 1e-6

This normalization is mandatory in the main protocol.

2.3 Mean-common

For a fixed timestep t:
A_common,mean^{(t)}(x) = (1/28) * sum_{i=1}^{28} A_tilde_{i,t}(x)

Corresponding residual:
B_mean_{i,t}(x) = A_tilde_{i,t}(x) - A_common,mean^{(t)}(x)

This is the main/common definition used for the principal report.

2.4 Fixed-rank truncated-SVD common

For each image x, stack all normalized activations across all layers and timesteps:
M(x) in R^{(28 * 10 * P) x 1152}

Compute truncated SVD:
M(x) approx U_K Sigma_K V_K^T

Use fixed ranks:
K in {16, 32, 64}

Then define:
A_common,tsvd,K_{i,t}(x) = A_tilde_{i,t}(x) V_K V_K^T

Residual:
B_tsvd,K_{i,t}(x) = A_tilde_{i,t}(x) - A_common,tsvd,K_{i,t}(x)

Important:
For the main protocol, DO NOT use adaptive-rank subspace-common.
Use only:
- mean-common
- fixed-rank truncated-SVD common with K = 16, 32, 64

TASK 0. EXTRACT ACTIVATIONS AND BUILD COMMON / RESIDUAL
------------------------------------------------------------------------------

Purpose
-------
This is the foundation task.
If Task 0 is wrong, all subsequent tasks become unreliable.

The goals are:
- extract the correct activations from SiT-XL/2 at every required layer;
- construct the two mandatory common families:
  1. mean-common
  2. fixed-rank truncated-SVD common with K in {16, 32, 64}
- construct the corresponding residual tensors correctly;
- save all tensors to disk with complete metadata so later tasks only need to load them.

Required preparation
--------------------
- Backbone: SiT-XL/2
- Number of blocks: exactly 28
- Hidden dimension: exactly 1152
- Checkpoint: the main checkpoint c*
- Seed: the main seed s0
- Main dataset: exactly 500 images
- Control dataset: exactly 100 images
- Timesteps: exactly the 10 selected timesteps
  T = {t_1,...,t_10}

Exact setting that must be used
-------------------------------
- Use all 28 layers:
  i in {1,...,28}
- Use all 10 timesteps:
  t in T
- Extract both:
  - patch embedding output
  - output after each transformer block
- Before building common, L2-normalize each patch token:
  A_tilde_{i,t}(x)_p = A_{i,t}(x)_p / (||A_{i,t}(x)_p||_2 + eps), eps = 1e-6
- Mean-common must be:
  A_common,mean^{(t)}(x) = (1/28) * sum_{i=1}^{28} A_tilde_{i,t}(x)
- Mean residual must be:
  B_mean_{i,t}(x) = A_tilde_{i,t}(x) - A_common,mean^{(t)}(x)
- For truncated-SVD common:
  M(x) in R^{(28*10*P) x 1152}, M(x) approx U_K Sigma_K V_K^T
  with K in {16, 32, 64}
- Truncated-SVD common must be:
  A_common,tsvd,K_{i,t}(x) = A_tilde_{i,t}(x) V_K V_K^T
- Truncated-SVD residual must be:
  B_tsvd,K_{i,t}(x) = A_tilde_{i,t}(x) - A_common,tsvd,K_{i,t}(x)
- Dtype:
  - bfloat16 or float16 for stored tensors
  - float32 for tensors used in PCA / SVD / CKA / wavelet

Input
-----
There is no tensor input from earlier tasks.
Direct inputs are:
- image x
- timestep t
- SiT-XL/2 at checkpoint c*
- seed s0

Steps to execute in order
-------------------------
1. For each image x, run the model at every timestep t in T.
2. Use forward hooks to extract:
   A_patch(x), A_{1,t}(x), ..., A_{28,t}(x)
3. Normalize all patch tokens to obtain A_tilde_{i,t}(x).
4. Build mean-common:
   A_common,mean^{(t)}(x)
5. Build corresponding mean residual:
   B_mean_{i,t}(x)
6. For each image x, build the large matrix:
   M(x) in R^{(28*10*P) x 1152}
7. For each K in {16,32,64}, run truncated SVD to obtain V_K.
8. Build:
   A_common,tsvd,K_{i,t}(x)
   B_tsvd,K_{i,t}(x)
9. Save all tensors to disk with metadata.

Core formulas
-------------
A_tilde_{i,t}(x)_p = A_{i,t}(x)_p / (||A_{i,t}(x)_p||_2 + eps)

A_common,mean^{(t)}(x) = (1/28) * sum_{i=1}^{28} A_tilde_{i,t}(x)

B_mean_{i,t}(x) = A_tilde_{i,t}(x) - A_common,mean^{(t)}(x)

M(x) approx U_K Sigma_K V_K^T,   K in {16,32,64}

A_common,tsvd,K_{i,t}(x) = A_tilde_{i,t}(x) V_K V_K^T

B_tsvd,K_{i,t}(x) = A_tilde_{i,t}(x) - A_common,tsvd,K_{i,t}(x)

Exact figure / table to generate
--------------------------------

Split / averaging order
-----------------------

Exact setting that must be used

Hyperparameters:
- similarity kernel: cosine similarity
- spatial distance: Manhattan distance on the patch grid
- LDS: r_near = r_far = H/2
- CDS: delta in {1,...,H-1}
- RMSC: L2-normalize tokens with eps = 1e-6
- SRSS:
  - if segmentation/semantic masks are available, use them
  - otherwise SRSS may be omitted from the main table but document that it requires masks

Input tensor names
------------------
- A_raw[i,t,x]
- A_common_mean[t,x]
- B_res_mean[i,t,x]
- A_common_tsvd_K16[i,t,x]
- A_common_tsvd_K32[i,t,x]
- A_common_tsvd_K64[i,t,x]
- B_res_tsvd_K16[i,t,x]
- B_res_tsvd_K32[i,t,x]
- B_res_tsvd_K64[i,t,x]

Steps to execute in order
-------------------------
1. For each tensor X, build the self-similarity matrix using cosine.
2. Compute LDS.
3. Compute CDS.
4. Compute RMSC.
5. If masks are available, compute SRSS.
6. Save results per-image / per-layer / per-time.
7. Average over all images.
8. Report separately for mean-common and each K.

Core formulas
-------------
Self-similarity:
K_X(t,t') = <x_t, x_t'> / ( ||x_t||_2 ||x_t'||_2 )

LDS:
LDS(X) = E[ K_X(t,t') | d(t,t') < r_near ] - E[ K_X(t,t') | d(t,t') >= r_far ]

Spatial correlogram:
g_X(delta) = E[ K_X(t,t') | d(t,t') = delta ]

Fit:
g_X(delta) approx alpha + beta * delta

CDS:
CDS(X) = - beta_hat

RMSC:
x_hat_t = x_t / (||x_t||_2 + eps)
x_bar = (1/T_tokens) * sum_t x_hat_t

RMSC(X) = sqrt( (1/T_tokens) * sum_t ||x_hat_t - x_bar||_2^2 )

SRSS:
SRSS(X) = E[ cos(x_anchor, x_pos) - cos(x_anchor, x_neg) ]
where anchor and pos are in the same semantic region and neg is outside the region.

------------------------------

- task4_mean_rmsc.csv

TASK 5. LAYER-ID / TIMESTEP-ID PROBES
------------------------------------------------------------------------------

Purpose
-------
Directly measure source-specific information in raw/common/residual.

Required preparation
--------------------
- Task 0 must be completed first
- Backbone: SiT-XL/2
- Checkpoint: c*
- Seed: s0
- Dataset: 500 images
- Use all 28 layers and all 10 timesteps

Exact setting that must be used
-------------------------------
This task must be run for:
- mean-common
- truncated-SVD common K = 16, 32, 64

Descriptor:
- use mean pooling over tokens only
- do not use any other pooling type

Labels:
- y_layer = i
- y_time = t

Input tensor names
------------------
- z_raw[i,t,x]
- z_meancommon[i,t,x]
- z_resmean[i,t,x]
- z_tsvdK16[i,t,x]
- z_tsvdK32[i,t,x]
- z_tsvdK64[i,t,x]
- z_restsK16[i,t,x]
- z_restsK32[i,t,x]
- z_restsK64[i,t,x]

Steps to execute in order
-------------------------
1. Build descriptors:
   z_{i,t}(x) = (1/P) * sum_{p=1}^{P} A_{i,t}(x)_p
2. Build descriptors for mean-common and mean-residual.
3. Build descriptors for each truncated-SVD common and residual.
4. Split by image:
   - train: 80%
   - test: 20%
5. Train a linear classifier for layer-id.
6. Train a linear classifier for timestep-id.
7. Repeat for raw and each common/residual family.

Core formula
------------

TASK 10 / TASK 11 IN SOURCE. WAVELET DECOMPOSITION
------------------------------------------------------------------------------

Purpose
-------
Analyze common and residual under a multiscale / frequency-like view using wavelets.

Required preparation
--------------------
- Task 0 must be completed first
- Backbone: SiT-XL/2
- Checkpoint: c*
- Seed: s0
- Dataset: 500 images

Exact setting that must be used
-------------------------------
This task must be run for:

- truncated-SVD common K = 16, 32, 64

Wavelet setting:
- wavelet family: Haar
- number of decomposition levels: exactly 2
- apply to:
  - A_common^{(t)}
  - B_{i,t}
- do not apply to A_patch in the main result

Input tensor names
------------------
- A_common_mean[t,x]
- B_res_mean[i,t,x]
- A_common_tsvd_K16[i,t,x]
- A_common_tsvd_K32[i,t,x]
- A_common_tsvd_K64[i,t,x]
- B_res_tsvd_K16[i,t,x]
- B_res_tsvd_K32[i,t,x]
- B_res_tsvd_K64[i,t,x]

Steps to execute in order
-------------------------
1. Reshape each tensor to:
   H_p x W_p x 1152
2. For each channel d, run 2D DWT at levels l = 1, 2.
3. Compute approximation energy.
4. Compute detail energy.
5. Compute the ratio:
   R_wave^{(l)}(Z) = E_A^{(l)}(Z) / E_detail^{(l)}(Z)
6. Report separately for mean-common and each K.

Core formulas
-------------
( A_d^{(l)}, H_d^{(l)}, V_d^{(l)}, D_d^{(l)} ) = DWT2^{(l)}( Z[:,:,d] )

E_A^{(l)}(Z) = sum_{d=1}^{1152} ||A_d^{(l)}||_F^2

E_H^{(l)}(Z) = sum_d ||H_d^{(l)}||_F^2
E_V^{(l)}(Z) = sum_d ||V_d^{(l)}||_F^2
E_D^{(l)}(Z) = sum_d ||D_d^{(l)}||_F^2

E_detail^{(l)}(Z) = E_H^{(l)}(Z) + E_V^{(l)}(Z) + E_D^{(l)}(Z)

R_wave^{(l)}(Z) = E_A^{(l)}(Z) / E_detail^{(l)}(Z)

Required output artifact names
------------------------------
- task11_wavelet_mean.csv
- task11_wavelet_tsvd.csv
- task11_wavelet_ratio.csv
- task11_figures.pdf

Exact figure / table to generate
--------------------------------
- Table T12: wavelet energy summary
- Figure T24: approximation vs detail energy
- Figure T25: layer-wise residual wavelet profile

Split / averaging order
-----------------------
- No split
- Compute wavelet energy per-image / per-layer / per-time first
- For common: average across images, then across timesteps
- For residual: average across images, then across timesteps, or keep layerwise if required by the figure
- Never mix mean-common and truncated-SVD common

Fail conditions / sanity checks
-------------------------------
- If energy is negative or NaN -> fail
- If common and residual produce identical wavelet profiles -> fail
- If reshape is wrong and yields abnormal shape -> fail
